workspacer

a tiling window manager for Windows 10

Download View on Github

portable zip file | msi installer - workspacer automatically checks for updates

configurable

configured entirely with C#; you can customize every piece, and change any behavior!

fast

manipulates windows instantly, and doesn't add any significant overhead to Windows

compatible

doesn't use DLL injection to manipulate windows, so it less likely to break things!

is this ready for use yet?

kinda! there are likely some major bugs in workspacer that I have not fixed yet, but it is more or less functionally complete. i am using workspacer 24/7 at home and at work at the moment, so it suites my needs (aside from the occasional hickup or crash). if you run into any problems, please open an issue!
